###Features
1. Configurable AI Scoring
Define weighted criteria (e.g., 5+ years of experience = 30%, specific tech stack = 40%, degree requirements = 20%).
Support hard filters (must-have requirements) and soft scoring (rank-based evaluation).
Transparent scoring breakdown per candidate.

2. Automated Shortlisting
Instantly ranks applicants based on defined constraints.
Flags top candidates for review.

3. Automated Communication
Send acceptance, rejection, or next-step emails automatically.

4. Bias Controls & Auditability
Criteria-based evaluation only (no hidden variables).
CV AI injection detection
